# 派派工房 電商平台

一個以 React 製作的甜點電商平台，提供使用者完整的購物體驗，包含商品瀏覽、購物車管理、優惠碼折扣、結帳流程等；同時也提供後台介面供管理員操作商品、折價券與訂單。

# 🚀 功能介紹

## 前台使用者

- 商品列表分類（蛋糕、派、鹹派）
- 商品詳情與加入購物車
- 優惠碼折扣功能
- 結帳流程與付款模擬
- 訂單查詢頁面
- AOS、Swiper 等動畫與輪播效果

- 商品 CRUD 功能（含 Modal 編輯視窗）
- 訂單查看與到貨日排序
- 資料表單驗證與回饋提示

# 🛠️ 技術棧

- **前端框架**：React 18
- **樣式庫**：Bootstrap 5、SCSS
- **動畫工具**：Swiper.js、AOS
- **表單處理**：React Hook Form
- **狀態管理**：React Context API
- **路由管理**：React Router v6
- **API 處理**：Axios
- **後端來源**：六角 API 或自建後端
